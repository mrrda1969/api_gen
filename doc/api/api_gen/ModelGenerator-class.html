ModelGenerator

ModelGenerator class

A generator for creating Dart model classes from schema definitions.

The ModelGenerator takes a schema (as a Map<String, dynamic>) and generates Dart model files in the specified outputDir. It handles schema parsing, error handling, and file generation, supporting nested models and various property types.

Example usage:

final generator = ModelGenerator('lib/models');
final result = generator.generate(schemaMap);
if (result.isSuccess) {
  print('Models generated successfully!');
}

Throws SchemaValidationException, FileOperationException, or CodeGenerationException for various error conditions.
